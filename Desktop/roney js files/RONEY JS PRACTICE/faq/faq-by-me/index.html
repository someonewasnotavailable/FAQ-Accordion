FAQs

What is frontend mentor and how will it help me?

Frontend Mentor offers realistic coding challenge to help devlopers improve their frontend coding skills with projects in HTML, CSS and Javascript. Suitable for all levels and ideal for portfolio building.

Is Frontend Mentor free?

Yes, Frontend Mentor is free to use, offering a large selection of free coding challenges, design assets, and assets for building projects. While a premium subscription exists for advanced challenges and design files (like Figma or Sketch), beginners can complete many "free" and "free+" challenges without paying

Can I use Frontend Mentor projects in my portfolio?

Yes, you can use Frontend Mentor projects in your portfolio to showcase your skills, particularly for junior-level roles

How can I get help if I'm stuck on a challenge?

If you are stuck on a Frontend Mentor challenge, the best way to get help is by joining their Discord server and posting in the #help channel.
